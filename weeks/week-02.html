第一節軟體

Illustrator 軟體基礎操作

介面 · 向量觀念 · 照片描圖 · 鋼筆工具

本節為 Illustrator 入門，目標是讓學生在進入實作前，先熟悉軟體介面與向量繪圖的核心觀念，建立後續描圖、繪製說明圖、輸出雷切檔的基礎。

A · 認識介面與向量觀念

🎯 教學目標

認識 Adobe Illustrator 的工作介面（工具列、控制列、屬性面板、圖層面板），理解向量圖與點陣圖的本質差異，並能獨立完成新檔建立、文件設定、儲存與輸出的基本流程。

📋 操作流程
  1. 介紹介面四大區塊：左側工具列、上方控制列、右側面板群、中央畫布。
  2. 練習新建文件並設定畫板尺寸、單位（mm）、色彩模式（RGB／CMYK）。
  3. 認識選取工具（V）、直接選取工具（A）、縮放（Z）、手形（H）等基本檢視操作。
  4. 練習圖層建立、命名、鎖定與顯示切換。
  5. 儲存為 .ai 原生檔，並另存為 .svg／.pdf 等輸出格式。
★ 教學重點

強調「向量圖無解析度限制」「以圖層分類管理零件」兩大核心觀念，並建立「邊做邊存檔」的工作習慣，避免當機損失。

B · 照片描圖

🎯 教學目標

學會將實體拍攝的照片或點陣圖匯入 Illustrator，透過「影像描圖（Image Trace）」自動轉換或手動鋼筆描繪，產出可縮放、可編輯的向量路徑，作為後續雷切與 3D 建模的圖檔基礎。

📋 操作流程
  1. 使用「檔案 → 置入（Shift + Ctrl + P）」將照片匯入畫布，建議勾選「連結」以減少檔案肥大。
  2. 將照片鎖定於獨立圖層作為描圖底圖，並調整不透明度至 50% 方便辨識。
  3. 選取照片後點上方控制列「影像描圖」下拉，嘗試「高保真度照片 / 低保真度照片 / 黑白標誌 / 線條圖」等預設集，比較結果。
  4. 點「展開（Expand）」將描圖結果轉為可編輯的向量路徑。
  5. 進階：新建空白圖層，用鋼筆工具沿照片輪廓手動描繪，產出更精準的向量線稿。
★ 教學重點

引導學生比較「自動描圖」與「手動描圖」的差異——自動快速但路徑點過多、輪廓粗糙；手動費時但路徑乾淨、可控性高。雷切用途的圖檔務必以手動描圖為主，避免自動產生的破碎節點造成雷射機停頓或重複切割。

影像描圖預設集適用情境
高保真度照片彩色寫實圖、輸出至印刷品。
低保真度照片風格化處理、降低細節。
黑白標誌LOGO、簡單剪影，雷切首選。
線條圖手繪稿、漫畫線條轉向量。
素描藝術／剪影創意風格化呈現。

C · 熟練鋼筆工具（Pen Tool）

🎯 教學目標

深入掌握鋼筆工具（快捷鍵 P）的核心操作：錨點新增與刪除、貝茲曲線控制把手的調整、直線段與曲線段的切換，以及複雜輪廓路徑的繪製。鋼筆工具是 Illustrator 最重要的基本功，也是區分「能用軟體」與「會用軟體」的關鍵分水嶺。

繪製直線段

選鋼筆工具（P）後，於畫布點擊第一個錨點，再點第二個，兩點自動連成直線段。按住 Shift 點擊可鎖定 45 度倍數方向。按 Enter 或切換工具可結束路徑；若要封閉，將最後一點接回起點（游標出現小圓圈提示）。

繪製曲線段

點擊產生第一個錨點後，按住滑鼠左鍵拖曳，會拉出兩條對稱「方向把手」，把手長度與角度決定曲線的曲率與方向。到下一個位置同樣點擊並拖曳，即可產生平滑 S 型曲線。把手越長曲率越大。

切換直線與曲線

畫曲線後若想接一段直線，按 Alt 點擊剛建立的錨點，可收回前進方向的把手，下一段即為直線。反之，從直線轉曲線則在最後一個錨點上按 Alt 拖曳拉出新把手。每個錨點的「進入把手」與「離開把手」可獨立控制，這是鋼筆工具最關鍵的觀念。

編輯既有路徑

新增錨點：游標移至路徑線段上變「鋼筆 +」即可點擊新增。刪除錨點：移至錨點上變「鋼筆 −」點擊刪除。轉換錨點：用錨點工具（Shift + C）點曲線錨點轉尖角、拖曳尖角轉平滑。移動錨點：用直接選取工具（A）點選後拖曳，方向鍵微調 1px（Shift 為 10px）。

封閉路徑與綜合練習

以照片零件為底圖，沿輪廓依序點擊或拖曳產生錨點，最後回到起點封閉路徑。封閉後的路徑才能套用填色與雷切輸出，未封閉路徑在雷切軟體中可能被忽略或產生錯誤。練習目標：以最少的錨點數量描出最平滑的輪廓——錨點越少，曲線越優雅，後續編輯也越容易。

第二節製圖

動手繪製製作說明圖

把實體物體轉化為清晰精確的加工說明圖

本節進入實務階段，學員將親手使用 Illustrator 製作精美的產品製作說明圖，以電子產品外殼拆解圖為範例，學習如何將實體物體轉化為清晰、精確的加工說明圖。

圖

本節四大學習要點

從拆解到應用的完整製圖思維
  • 零件拆解與排列：將完整產品拆解成各個獨立零件，以爆炸展開圖的方式有條理地排列，讓閱讀者一目了然各零件的形狀與對應關係。
  • 精確運用工具：以鋼筆工具與幾何圖形工具切實繪製每個零件外形，包含卡榫、孔洞、凹槽等細節，訓練對形狀的觀察力與線條的精準控制。
  • 線條規範與圖層管理：學習工程製圖的基本規範，統一線條粗細、色彩與風格，並以圖層分類管理不同零件，養成專業檔案整理習慣。
  • 說明圖的應用架構：完成的零件展開圖可應用於產品說明書、建造指南、專利申請附圖或製造規格文件，是工業設計與產品開發流程中的視覺溝通工具。

第三節雷切

雷切實驗製作與假組

切割後檢驗與迭代修正

本節完成最關鍵的實體驗證階段——將前兩節繪製的向量圖檔送入雷射切割機進行實際切割，並透過「假組」過程檢驗設計的密合度與精確度。

雷射切割前的文件準備

將 Illustrator 中不同的零件圖依雷切機規格輸出設定，包含線條顏色區分切割與刻線、確認尺寸比例為 1:1、排版省料等，讓圖檔能被機器正確讀取與執行。

實驗性切割

以實驗心態進行第一次切割，目的不求完美，而是透過實際材料（木板、壓克力）的切割結果，觀察零件尺寸是否吻合、卡榫是否能順利嵌合、間隙是否需要修改。

假組驗證

將切割完成的零件進行「假組」——即不上膠、不固定，純粹手動試拼初組，逐一檢查每個部件的相對位置與裝配度，找出設計上的疏漏。

發現問題 → 解決問題

假組過程中一旦發現問題，立即回 Illustrator 修改圖檔、重新驗證切割，形成「設計 → 製作 → 檢驗 → 修正」的迭代循環，培養解決實務問題的應變能力，這也是從圖紙走向最真實的學習歷程。
