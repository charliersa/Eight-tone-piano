第 10–11 週 · 試印與測試

電子鋼琴
測試與迭代修正

本兩週進入電子鋼琴外盒的「試印與測試」階段。承接第八、九週把外盒零件 3D 列印出來之後，這裡要逐件檢驗列印品質、實際假組試拼，並把電路板、喇叭、琴鍵按鈕放入測試是否干涉，找出問題後回頭修檔重印，確保進入正式組裝前每個零件都合格、能順利裝配。

兩週 · 3 節主軸 核心 測試 → 修正 → 再測試 工具：游標卡尺
第一節檢驗

列印件品質檢驗與篩選

列印完成不等於可用
🎯 教學目標

學會以系統性的方式檢查 3D 列印外盒零件的品質，判斷哪些可用、哪些需重印，建立「列印完成不等於可用」的品質意識。

📋 操作流程
  1. 逐件目視檢查表面：有無翹邊、缺層、拉絲、層紋過深等列印瑕疵。
  2. 以游標卡尺量測關鍵尺寸，比對設計圖是否在誤差範圍內。
  3. 檢查各開孔位置與大小——喇叭孔、按鈕孔、走線孔是否對齊、是否需擴孔。
  4. 將零件分為「合格」「需修整」「需重印」三類並貼標記錄。

操作解說：剛印出的零件常有翹邊、毛邊或尺寸偏差。先用眼睛看、用卡尺量，重點檢查喇叭孔與 8 個按鈕孔的位置與大小是否正確——這些孔位若偏差，後面電路與琴鍵就裝不進去。把零件分成三類，合格的收好、需修的進第三節處理、需重印的記下問題回去修檔。

3

三色標籤分類法

  • 🟢 合格：尺寸、孔位、表面都 OK，直接收好待組裝。
  • 🟡 需修整：小瑕疵（毛邊、孔略緊），第三節打磨擴孔即可。
  • 🔴 需重印：尺寸或孔位偏差大，記下問題回去修檔重印。
第二節假組

外盒假組與機構干涉測試

先試裝、不上膠
🎯 教學目標

透過實際假組（不上膠的試拼）驗證外盒各面是否能順利接合，並把電路板、喇叭與琴鍵按鈕放入測試空間是否足夠、有無干涉。

📋 操作流程
  1. 依組裝順序將外盒各面卡榫試拼，確認接合緊密、不鬆脫也不過緊。
  2. 將電路板與喇叭放入盒內預定位置，確認固定孔對齊、空間足夠。
  3. 把琴鍵按鈕由面板孔位裝入，逐顆測試能否順利按壓、不卡邊。
  4. 記錄所有干涉、卡住或對不上的位置，作為修正依據。

操作解說：假組就是「先試裝、不上膠」，目的是在正式組裝前抓出問題。把外盒各面卡榫拼起來，再把電路板、喇叭、按鍵一一放進去，特別注意琴鍵按下去會不會卡到孔邊、電路板會不會頂到盒壁。哪裡裝不進、哪裡卡住，全部記下來，下一節集中處理。

第三節迭代

問題記錄與迭代修正

一次印不好很正常，重點是會修
🎯 教學目標

將前兩節找出的問題系統化整理成修正清單，學會回到設計檔案修改參數並重新列印，體會 3D 列印「設計—列印—測試—修正」的迭代循環。

📋 操作流程
  1. 彙整檢驗與假組階段的所有問題，列成「待修清單」。
  2. 區分問題類型——屬於列印參數（如翹邊、尺寸縮收）或屬於設計尺寸（如孔位偏移、間距太擠）。
  3. 回到 Tinkercad 或切片軟體修改對應參數，重新輸出列印檔。
  4. 重印問題零件後再次假組驗證，直到全部零件通過。

操作解說：這一節是 3D 列印最重要的觀念——「一次印不好很正常，重點是會修」。把問題分清楚是「印壞的」還是「設計錯的」：印壞的調列印參數重印就好；設計錯的要回到 Tinkercad 改尺寸或孔位再重印。改完再試組一次，反覆到所有零件都裝得上、按得動，這個「測試 → 修正 → 再測試」的循環就是創客的核心能力。

💡 拍照記錄進步

鼓勵學生把每次修正前後的零件並排拍照記錄，看見自己「一次比一次好」的進步，也建立面對失敗、持續改善的態度。
