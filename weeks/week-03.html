孔位、間距、幾何基礎

本週進入機構製作的核心基礎，從孔位定位、按鍵孔位陣列到零件間距，逐步建立精密機構設計的觀念。學生將透過 Illustrator 的精準繪圖與雷切實驗，理解「公差」與「咬合」在機構設計中的重要性，並學會用 doodle3D 軟體將 2D 圖檔延伸為 3D 模型，銜接後續的 3D 列印實作。

第一節定位

孔對孔的間距與定位

理解間距如何影響對位與組裝精度
🎯 教學目標

讓學生理解孔位之間的間距如何影響機構的對位與組裝精度，掌握 Illustrator 中「對齊」「均分間距」「智慧型參考線」等定位工具。

📋 操作流程
  1. 講解標準孔徑與間距規格。
  2. 在 Illustrator 中以「變形面板」輸入精確座標繪製孔位。
  3. 輸出小型試片進行雷切。
  4. 以實物量測比對設計值與切割結果的誤差。
★ 教學重點

強調「設計尺寸 ≠ 雷切尺寸」的觀念，引導學生記錄雷射光束的耗損值（kerf），作為日後設計補償的依據。

K

名詞小教室：什麼是 Kerf（切口寬度）？

雷射光束本身有寬度

雷射切割時，光束會「燒掉」一條約 0.1–0.2mm 寬的材料，這條被燒掉的寬度就叫 kerf。它會讓切出來的孔比設計值略大、凸出的卡榫比設計值略小。理解 kerf，才能在設計時預留補償，做出剛好咬合的零件。

第二節陣列

琴鍵按鈕孔位陣列與定位

8 顆琴鍵的等距排列與對位
🎯 教學目標

延續第一節「孔對孔的間距與定位」，本節聚焦電子鋼琴最關鍵的機構——琴鍵按鈕的孔位陣列。引導學生理解 8 顆琴鍵如何在面板上等距排列、對齊下方電路按鈕，建立「陣列、間距、對位」的幾何基礎。

📋 操作流程
  1. 量測單顆按鈕的外徑與所需開孔尺寸，確認面板開孔比按鈕略大、能順利穿過。
  2. 決定琴鍵間距（孔中心到孔中心的距離），讓相鄰按鍵不互相干涉、手指好按。
  3. 以等距陣列規劃 8 個孔位，必要時分成兩排或加大白鍵、黑鍵的相對位置。
  4. 標出每個孔位的中心座標，作為下一節電腦繪製面板的依據。
幾

三個幾何重點

陣列 · 對位 · 間距與手感
  • 陣列（Array）：8 個孔不是隨意排，而是「等間距重複」的陣列。只要定好「起點 + 間距 + 數量」，整排孔位就能精準產生——這正是後續在 Illustrator 用「再次變形（Ctrl+D）」快速複製孔位的概念。
  • 對位（Registration）：面板上的按鈕孔必須對齊下方電路板上的實體按鈕。孔位一旦偏移，按鍵就會卡住或按不到，因此「孔對孔」的對位精度是這台鋼琴能不能彈的關鍵。
  • 間距與手感：琴鍵太擠會誤觸、太鬆會浪費面板空間。引導學生實際把手指放上去比畫，找出好按又緊湊的間距，體會「功能與尺寸」如何互相牽制。

操作解說

這一節把第一節的「孔對孔定位」放大到整排琴鍵。先量一顆按鈕、定好開孔大小與間距，再用等距陣列排出 8 個孔位並記下座標。重點是讓學生明白：面板上每個孔都要對準下方電路的按鈕，這個對位關係決定了琴鍵按下去能不能正常發聲。

第三節立體

doodle3D 與 Bambu Studio 概念組合

從 2D 平面到 3D 立體的第一步
🎯 教學目標

學習使用 doodle3D 將 2D 平面草圖轉換為 3D 立體模型，並認識 Bambu Studio 的切片邏輯，完成從平面到立體的設計轉換。

📋 操作流程
  1. 以 doodle3D 拉伸平面圖形產生厚度。
  2. 將 2D 零件圖延伸為 3D 爆炸圖。
  3. 匯出 STL 檔。
  4. 在 Bambu Studio 中載入並設定基本列印參數。
★ 教學重點

讓學生親自體驗「2D 思維」與「3D 思維」的差異，理解平面圖在轉為立體時厚度、方向、支撐等新增的設計變因。
