第一節檔案

課程歷程與學習檔案整理

記錄也是創客素養
🎯 教學目標

引導學生回顧十五週的製作歷程，將拍攝紀錄、草圖、向量檔、3D 模型與成品照片彙整成一份完整的學習檔案，理解「記錄」本身也是創客素養的一環。

📋 操作流程
  1. 依週次分類整理歷程照片與螢幕截圖。
  2. 蒐集各階段檔案（AI 向量檔、Tinkercad 模型、列印參數）。
  3. 以資料夾與命名規則建立可追溯的檔案結構。
  4. 挑選每週最具代表性的一張成果照作為亮點。

操作解說：把散落在手機、電腦與雲端的素材集中起來，按週次建立資料夾並統一命名。整理的過程也是再次檢視自己學習軌跡的機會，為後續書面報告與發表預備素材。

第二節報告

書面報告與簡報製作

一頁一重點、圖文並陳
🎯 教學目標

將整理好的歷程素材轉化為結構清晰的書面報告與簡報，練習以文字與圖像說明設計理念、製作流程與遇到的問題。

📋 操作流程
  1. 擬定報告架構：動機、設計、製作、成果、反思。
  2. 將每階段照片搭配 2～3 句說明文字。
  3. 製作簡報，每頁以一個重點搭配一張圖。
  4. 互相檢閱，調整文字精簡度與版面一致性。

操作解說：報告不必長，重點是把「為什麼這樣做、怎麼做、結果如何」講清楚。一頁一重點、圖文並陳，讓沒看過製作過程的人也能看懂作品的價值。

構

五段報告架構範本

  • 動機：為什麼想做這台鋼琴？想解決或表達什麼？
  • 設計：外盒造型、在地元素、機構配置怎麼決定的？
  • 製作：向量、建模、雷切、列印、電路的關鍵步驟。
  • 成果：完成的作品照與功能展示。
  • 反思：遇到最大的困難、怎麼解決、學到什麼。
第三節展示

作品成果整理與展示準備

模擬展覽現場
🎯 教學目標

完成實體作品的最後檢查與展示佈置，將電子鋼琴成品、製作說明圖與學習檔案整合為一套可展出的成果，培養作品完整呈現的意識。

📋 操作流程
  1. 檢查作品外觀、結構與發聲功能是否正常。
  2. 製作作品標示卡（名稱、創作者、設計概念）。
  3. 規劃展示桌面動線與作品擺放方式。
  4. 試擺一次並拍攝成果照存檔。

操作解說：把作品、說明圖與標示卡擺在一起，模擬展覽現場。先試擺、試彈、試講，確認一切就緒，也讓下週的公開發表更有把握。
