認識 Arduino

本週將前幾週製作的木製機構作品與電子互動結合，認識 Arduino 微控制器的基本概念。學生透過製作「鋼琴聲音按鈕」入門專題，學習如何讓按鈕的按壓觸發聲音輸出，建立「感測器輸入 → 程式判斷 → 致動器輸出」的互動裝置基本架構，為後續創客作品的智慧化奠定基礎。

第一節電路

認識 Arduino 與電路基礎

先認識這顆「大腦」與它的接線方式
🎯 教學目標

認識 Arduino 開發板的核心組件（數位／類比腳位、電源、USB 接口），理解麵包板的內部接線邏輯與基本電路概念（電壓、電流、接地）。

Arduino 是一塊「可程式化的電路控制板」，可以把它想成電子作品的大腦。我們用電腦寫好程式上傳給它，它就能依指令讀取感測器（輸入）、控制燈光與聲音（輸出）。

Arduino UNO 板各部位功能

USB 接口：連電腦上傳程式與供電（5V）。電源接口（DC Jack）：接 7–12V 變壓器獨立運作。數位腳位（0–13）：只處理 HIGH（5V）／LOW（0V）兩種狀態，接按鈕、LED、蜂鳴器；標「~」的腳位（3、5、6、9、10、11）支援 PWM。類比輸入（A0–A5）：讀取連續數值 0–1023。電源腳位：5V、3.3V 與 GND（接地，務必與外部零件共地）。內建 LED（腳位 13）：測試板子是否正常最快的方法。

教學提示：請學生親手指認 USB、電源、GND、腳位 13 四個關鍵位置，這是後續所有接線的基準。

麵包板的連通規則

兩側電源軌（紅藍長條）：沿長邊「整條縱向相通」，紅（+）接電源、藍（−）接 GND。中間插孔區：每一「直行」的 5 個孔彼此相通，但橫向不相通；中央凹槽左右隔開。

接線口訣：同一直行＝相通，跨過中央溝＝不通。先把 5V 與 GND 各拉一條到電源軌，之後零件就近取電，線路最乾淨。

完成第一個 LED 點亮電路

LED 極性：長腳為正極（接電源）、短腳為負極（接 GND）。限流電阻：LED 直接接 5V 會燒毀，必須串聯 220Ω–330Ω 電阻。接線：腳位 13 → LED 長腳 → LED 短腳串接 220Ω 電阻 → 電阻另一端接 GND。

教學提示：先不通電完成接線，由老師檢查極性與限流電阻無誤後再插 USB，建立「先檢查再通電」的習慣。

上傳 Blink 範例驗證環境

開啟 Arduino IDE →「工具 → 開發板」選 Arduino UNO →「工具 → 連接埠」選對應 COM 埠 →「檔案 → 範例 → 01.Basics → Blink」→ 點左上角「→（上傳）」。成功後腳位 13 的 LED 會每秒閃爍一次。程式結構：setup() 只執行一次、loop() 不斷重複。試把 delay(1000) 改成 delay(200) 觀察閃爍變快——這是學生第一次體會「改程式 → 看到實體改變」的成就感。

!

常見問題排除

找不到連接埠檢查 USB 線是否可「傳輸資料」、是否需安裝 CH340 驅動。
上傳失敗確認開發板型號與連接埠正確，拔除接在腳位 0、1 的線。
LED 不亮檢查極性、限流電阻是否鬆脫、是否插在相通的同一直行。
★ 教學重點

建立「硬體接線」與「軟體程式」必須同步正確的觀念，並強調極性、限流電阻等基本電路安全常識。

第二節發聲

按鈕輸入與蜂鳴器發聲

Arduino 第一次「發出聲音」
🎯 教學目標

學會以數位輸入讀取按鈕狀態，並透過 tone() 函式驅動蜂鳴器發出不同頻率的音階，建立「輸入 → 判斷 → 輸出」的程式流程。

件

認識零件：按鈕與蜂鳴器

  • 按鈕（Push Button）：四隻腳的輕觸開關，對角兩腳在按下時導通。按下＝接通、放開＝斷開。
  • 蜂鳴器（Buzzer）：分「有源」與「無源」。有源通電固定叫一個音、無法控制音高；要彈出 Do、Re、Mi，必須用無源蜂鳴器搭配 tone() 控制頻率。本節請使用無源蜂鳴器。

接上按鈕並配置下拉電阻

什麼是下拉電阻：腳位沒接東西時電位會「浮動」，讀值忽高忽低。下拉電阻（10kΩ）把腳位平時固定拉到 GND（讀 LOW），按下時才接 5V（讀 HIGH）。接線：按鈕一腳接 5V，另一對角腳同時接「數位腳位 2」與「一顆 10kΩ 電阻到 GND」。

簡化做法：可改用程式內建上拉電阻——按鈕一腳接 GND、另一腳接腳位 2，程式以 INPUT_PULLUP 設定（邏輯相反）。初學建議先用實體下拉電阻較好理解。

以 digitalRead() 偵測按壓

用 pinMode() 把腳位 2 設為輸入，再用 digitalRead() 讀狀態，搭配 if 判斷。先做最單純的「按下亮內建 LED」驗證接線：

button_led.ino
const int btn = 2;

void setup() {
  pinMode(btn, INPUT);
  pinMode(13, OUTPUT);
}

void loop() {
  if (digitalRead(btn) == HIGH) {   // 按下
    digitalWrite(13, HIGH);
  } else {
    digitalWrite(13, LOW);
  }
}

連接蜂鳴器並查表設定音階頻率

接線：無源蜂鳴器正極（長腳）接「數位腳位 8」，負極（短腳）接 GND。音高就是頻率：聲音的高低由「振動頻率（Hz）」決定，頻率越高聲音越尖。tone(腳位, 頻率) 發出指定頻率，noTone(腳位) 停止。

音階（中央 Do 起）DoReMiFaSolLaSi高 Do
頻率 (Hz)262294330349392440494523

把上面 LED 換成蜂鳴器發聲，即完成「按一下響一聲（Do）」：

button_buzzer.ino
const int btn = 2;
const int buzzer = 8;

void setup() {
  pinMode(btn, INPUT);
}

void loop() {
  if (digitalRead(btn) == HIGH) {
    tone(buzzer, 262);   // 發出 Do（262 Hz）
  } else {
    noTone(buzzer);      // 放開即停
  }
}

完成「按一下響一聲」並加上去彈跳

上傳後按下按鈕，蜂鳴器發出 Do、放開停止。去彈跳（debounce）：機械按鈕在按下瞬間有微小抖動，可能造成聲音斷續。最簡單的做法是在偵測到按下後加一行 delay(20) 略過抖動區間，聲音就會乾淨穩定。

!

常見問題排除

完全不發聲確認用的是「無源」蜂鳴器；檢查正負極與腳位是否接對。
聲音一直響多半是按鈕接線或下拉電阻沒接好造成腳位浮動，檢查下拉電阻是否接到 GND。
按一下響好幾聲按鈕彈跳所致，於判斷式後加 delay(20) 去彈跳。
音不準tone() 為方波近似音，與樂器音色不同屬正常；頻率正確音高即正確。
★ 教學重點

認識「音高 = 頻率」的物理關係，並引導學生理解程式邏輯中 if 判斷式如何對應實體世界的「如果……就……」。

第三節整合

鋼琴聲音按鈕機構整合

擴充成 8 鍵迷你鋼琴並裝進外殼
🎯 教學目標

將多顆按鈕對應不同音階，整合為迷你鋼琴互動裝置，並結合前幾週製作的雷切外殼，完成具完整外觀的電子作品。

擴充電路為 8 顆按鈕對應 Do 到高音 Do

接線規劃：8 顆按鈕分別接數位腳位 2～9，每顆比照第二節接一顆 10kΩ 下拉電阻到 GND；蜂鳴器移到腳位 10。所有按鈕與電阻共用麵包板兩側的 5V 與 GND 電源軌。

對應關係：腳位 2→Do、3→Re、4→Mi、5→Fa、6→Sol、7→La、8→Si、9→高音 Do。

教學提示：先一次只接 1～2 顆按鈕測通，確認無誤再逐顆擴充，比一次接 8 顆再除錯容易得多。

以陣列與 for 迴圈整理程式碼

為什麼用陣列：若每顆按鈕都寫一段 if，程式會又長又難改。把「腳位」與「頻率」各放進一個陣列，再用 for 迴圈一次檢查 8 顆按鈕，程式碼精簡且好擴充。

mini_piano_8keys.ino
// 8 顆按鈕腳位
const int btns[8]  = {2, 3, 4, 5, 6, 7, 8, 9};
// Do ~ 高音 Do 頻率
const int notes[8] = {262, 294, 330, 349, 392, 440, 494, 523};
const int buzzer = 10;

void setup() {
  for (int i = 0; i < 8; i++) {
    pinMode(btns[i], INPUT);
  }
}

void loop() {
  bool pressed = false;
  for (int i = 0; i < 8; i++) {
    if (digitalRead(btns[i]) == HIGH) {  // 哪顆被按下
      tone(buzzer, notes[i]);            // 就發對應音
      pressed = true;
    }
  }
  if (!pressed) noTone(buzzer);          // 都沒按就靜音
}

程式說明：for 迴圈逐一檢查 8 顆按鈕，按到哪顆就用同一索引 i 從 notes[] 取出對應頻率發聲；用 pressed 旗標記錄是否有任何按鍵被按下，全部放開時才靜音。日後想加按鍵或改音，只要改陣列內容即可。

將電路安裝至雷切外殼內，按鈕對齊面板開孔

面板開孔：回 Illustrator 依按鈕實際間距排 8 個圓孔（常見 Ø12mm），間距建議 18～20mm 以利手指分開按壓。固定零件：按鈕穿過面板孔後以螺帽鎖固；Arduino 與麵包板用 3D 列印固定座（PETG）鎖在底板上。走線整理：USB 線預留從外殼側邊出線的開口。

呼應第五週選材：面板裝飾件用 PLA、需鎖螺絲承力的固定座用 PETG、軟觸感琴鍵墊可用 TPU。

測試 Do Re Mi 聲音並驗收

逐鍵測試：從 Do 到高音 Do 逐顆按壓，確認每顆發出正確且唯一的音。彈一首驗收：以「小星星」開頭為例——Do Do Sol Sol La La Sol，能完整彈出即代表電路、程式、外殼三者整合成功。延伸挑戰：加裝 LED 隨按鍵亮燈，或用陣列再存一首樂曲讓裝置自動演奏。

★ 教學重點

剖析「電子鋼琴」的發聲原理，引導學生比較機械式（凸輪撥動）與電子式（程式控制）兩種發聲機制的異同，串連整學期的學習脈絡。
