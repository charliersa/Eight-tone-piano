彈性緩衝週
檔案除錯與 Maker Lab

本週為彈性緩衝週。前幾週從建模、雷切到 3D 列印，學生的檔案常會出現大小不一的問題（路徑未封閉、尺寸不符、模型錯誤、列印失敗等），需要時間來回修正；因此預留這一週讓進度落後或檔案有狀況的學生補做、重印，確保每個人都能跟上。進度超前的學生則進入 Maker Lab 創客實驗室延伸體驗。

第一節除錯

檔案問題盤點與排除

先檢視卡在哪個環節，再對症處理

本節聚焦檔案問題的盤點與排除：先協助學生檢視自己卡在哪個環節，再對症處理。採個別指導為主，老師逐一檢視每位學生的檔案與成品狀態，列出待修清單；已完成的學生可協助同學或進入第三節的 Maker Lab。

★ 緩衝週的價值

在「不讓任何人掉隊」——把前幾週累積的小問題一次清掉，並讓學生在反覆修正中真正內化「設計 → 製作 → 檢驗 → 修正」的迭代思維。

第二節補做

個別補做與重印

逐一檢視、列出待修清單
🎯 教學目標

讓進度落後或檔案有狀況的學生，在老師個別指導下完成補做與重印，使每位學生手上都有一套可用、可組裝的零件，為後續組裝週做好準備。

📋 進行方式
  1. 老師逐一檢視每位學生的檔案與成品狀態。
  2. 列出個人「待修清單」，標註屬於哪一類問題。
  3. 學生依清單回到對應軟體修正、重新輸出與重印。
  4. 已完成的學生協助同學，或進入第三節 Maker Lab。
💡 同儕互助

讓已完成的學生當「小老師」協助卡關同學，既鞏固自己的理解，也減輕老師個別指導的負擔，營造創客社群的互助氛圍。

第三節延伸

Maker Lab 創客實驗室

把切片軟體當成探索與創作的工具

本節為進度超前學生的延伸體驗：在外盒列印與修正都完成、還有餘裕的情況下，帶他們玩 Bambu Studio 的 Maker Lab，把切片軟體當成探索與創作的工具，而不只是「按 Slice」的步驟。

L

什麼是 Maker Lab？

Maker Lab 是 Bambu Studio 內建的創意功能集，提供文字浮雕、圖片轉浮雕（Image to Relief／litho）、QR Code、徽章、聲波圖等模板，讓學生不需從零建模就能快速產出有趣的個人化作品。

🔤

文字 3D 字牌

為自己的電子鋼琴做一塊姓名牌或標語。

🖼️

圖片浮雕

把喜歡的圖案轉成可列印的浮雕板。

🔳

QR Code 牌

連到自己的作品介紹或演奏影片。

🔊

聲波圖

把一段旋律的波形列印成裝飾，呼應「聲音」主題。

📋 操作方式
  1. 於 Bambu Studio 上方功能列找到 Maker Lab。
  2. 選擇模板，輸入文字或匯入圖片。
  3. 調整尺寸與厚度，軟體會自動產生 3D 模型。
  4. 依前幾週的切片流程列印即可。
★ 教學重點

這一節是「玩中學」——鼓勵學生自由嘗試、做出與電子鋼琴搭配的小裝飾或個人化配件，把學到的列印技能用在自己想做的東西上，深化對工具的掌握與創作的成就感。

💡 本週教學提示

緩衝週是課程節奏的「呼吸點」。它的存在本身就是一種教學設計——承認失敗與修正是創客的常態。建議老師大方告訴學生：「印壞、做錯都正常，會修才是真本事。」讓超前與落後的學生各得其所。
