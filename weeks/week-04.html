第一節匯入

從 Illustrator 匯入圖檔至 Tinkercad

SVG 是 2D → 3D 之間的橋樑格式
🎯 教學目標

學會將 Illustrator 繪製的向量圖形（SVG 格式）匯入 Tinkercad，作為 3D 造型的底圖或拉伸基礎。

在 Illustrator 將圖形輸出為 SVG

為什麼是 SVG：SVG（Scalable Vector Graphics）以「數學路徑」描述形狀，與 JPG／PNG 等像素點陣圖不同。Tinkercad 只能讀取「封閉路徑」的向量輪廓，再沿 Z 軸拉高為立體模型，因此 SVG 是 2D → 3D 的標準橋樑格式。

檢查路徑封閉：按 Ctrl + Y 切換輪廓檢視，確認所有要拉伸的形狀皆為「頭尾相連的封閉路徑」；未封閉的路徑匯入後無法產生厚度，只會呈現一條線。

文字建立外框：若圖檔含文字，選取後按 Ctrl + Shift + O 轉為向量路徑。接著「檔案 → 另存新檔 → SVG」，SVG 設定檔選「SVG 1.1」、字體選「轉換成外框」、影像位置選「內嵌」。

常見錯誤：路徑未封閉導致無法拉伸；多路徑重疊時 Tinkercad 會把重疊區視為孔洞而破壞造型；單一 SVG 形狀過多（節點過密）會卡頓或閃退，建議單檔形狀控制在 10 個以內。

於 Tinkercad「匯入」面板載入 SVG 並設定尺寸與高度

進入 Tinkercad 編輯畫面 → 點右上角「匯入（Import）」（雲朵向上箭頭）→「從電腦選擇檔案」載入 SVG。

比例（Scale）：預設 1.0，Tinkercad 會把 SVG 數值解讀為公釐；若物件過大爆出工作平面，將比例調為 0.1 或 0.5。

高度（Height）：即 Z 軸拉伸厚度，例如面板設 3mm、結構支撐件設 5mm，匯入後仍可修改。

檢查尺寸：點選物件看右上角 X／Y／Z 數值；尺寸不符可直接輸入正確數字等比例縮放。

教學提示：匯入前先於 Illustrator 將畫板設為與實際零件 1:1（例如面板 80×80mm），匯入 Tinkercad 後尺寸通常即可吻合。

與內建幾何體組合修整

匯入的 SVG 物件與內建立方體、圓柱體屬性完全相同，可做布林運算，也可切換「實體 / 孔洞」屬性。

加厚補強（聯集）：面板太薄時拖入立方體置於底部，框選兩者按 Ctrl+G 群組，形成有底座的厚實零件。

挖機芯孔位（差集）：拖入圓柱按 H 切為「孔洞」（半透明斜線），移至打孔位置、確認孔洞高度大於面板厚度，框選後群組即挖出孔位。

對齊與鏡像：對齊工具（L）使物件於各軸對齊；卡榫成對出現時可先做一個按 Ctrl+D 複製、再按 M 鏡像翻面；正交檢視（數字鍵 1–6）切換六面檢查。確認後「匯出 → .STL」。

第二節建模

Tinkercad 基礎幾何組合教學

實體 + 孔洞 + 群組 = 多數工業零件
🎯 教學目標

掌握 Tinkercad 的核心操作邏輯——以「實體」與「孔洞」兩種屬性，透過群組（Group）達成布林運算的造型效果。

移動、旋轉、縮放基本操作

從右側「基本形狀」面板拖曳立方體至工作平面。移動：左鍵拖曳於 XY 平面；按 Alt 拖上方黑色錐形把手可升降 Z 軸。旋轉：物件外圍三個彎曲箭頭對應 X／Y／Z 軸，按 Shift 鎖定 22.5 度一階。縮放：白色方塊把手單軸縮放、角落黑色把手等比例。練習重點：請學生輸入精確數值（例如 20×20×10mm），培養「以數字思考」的習慣。

以聯集組合多個幾何體

拖曳兩個以上物件（如立方體 + 圓柱）→ 用對齊工具（L）擺好相對位置 → 框選後按「群組（Ctrl+G）」。群組後兩物件合而為一、顏色統一，這就是布林運算的「聯集（Union）」。需還原按 Ctrl+Shift+G 取消群組。

用「孔洞」屬性挖除材料製作卡榫孔

拖一個圓柱／立方體作為「要挖除的形狀」→ 右上角屬性由「實體」切為「孔洞（Hole）」（變半透明斜線）→ 移到主體要挖的位置（孔洞高度需 ≥ 主體厚度）→ 框選主體+孔洞按 Ctrl+G，孔洞體積即被扣除。這就是「差集（Subtract）」，是製作卡榫孔、螺絲孔、穿線孔的核心技巧。

完成一個簡易機構零件

綜合練習：製作「附螺絲孔的固定座」——立方體基座（30×30×8mm），中央以圓柱孔洞挖 Ø4mm 穿孔，四角再挖 Ø3mm 螺絲固定孔。可加導角（圓角立方體）或補強肋（細長立方體聯集於底部）。完成後按數字鍵 1–6 切換正交視圖檢查孔位對齊、無破面，最後「匯出 → .STL」。

第三節切片

Bambu Studio 與基本列印設定

設計與機器之間的橋樑
🎯 教學目標

認識 Bambu Studio 的工作介面，理解列印前所需的基本設定，包括擺放方位、層高、填充率與支撐結構。

將 Tinkercad 模型匯出為 STL

Tinkercad 右上角「匯出 → .STL」。多物件可勾「僅匯出選取的形狀」分件輸出。檔名建議「作品名_零件名_版次.stl」（例：電子鋼琴_面板_v2.stl），方便版本管理。

開啟 Bambu Studio 並匯入模型

確認左上角「印表機」設為實際機型（X1C / P1S / A1 mini）→ 右側「線材」面板選實際材料（PLA / PETG / ABS），軟體自動套用對應溫度與速度 → 拖曳 STL 至工作平台。

擺放方位與基本調整

移動（M）、旋轉（R，按 Shift 鎖 15 度）、縮放（S）；按 F 自動方位（Auto Orient）。教學重點：平整面朝下、避開大量懸空，可大幅減少支撐用量與失敗率。

列印參數設定（最關鍵的四項）

層高（Layer Height）：0.2mm 標準；外觀件 0.12mm、結構件 0.28mm。越薄越精細但時間倍增。
填充率（Infill）：標準 15%、結構件 25–40%、外觀件 10%。
外殼層數（Wall Loops）：預設 2 層；承重或鎖螺絲調 3–4 層。
支撐（Support）：懸空角度 45 度以上才啟用，可選一般 / 樹狀。

切片與預覽

右下角「切片（Slice）」→ 切到「預覽」分頁看預估時間、線材用量、總層數 → 拖右側層數滑桿逐層檢視，確認支撐位置、填充樣式、首層附著。發現問題回「準備」分頁修改後重新切片。

傳送至印表機

三種方式：① 雲端列印（WiFi 直送，最方便，需登入帳號）；② SD 卡（匯出 .3mf / .gcode.3mf）；③ LAN 區網（學校環境建議）。傳送前確認：平台清潔、線材充足、噴頭歸位。
