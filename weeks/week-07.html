放大版電子鋼琴
外盒設計與雷射切割

引導學生運用 Tinkercad 進行放大版電子鋼琴外盒的造型設計，從機構運轉所需的內部空間出發，思考音樂模組、Arduino 控制板與電池座的容置位置、出線孔與開關開口的配置，建立「形隨機能」的設計思維。過程中掌握尺寸測量、壁厚設定、卡榫與組裝公差等實務細節，並融入新營在地文化元素發想專屬造型。本週分為「設計篇」與「雷切篇」兩大段，共六節。

設計 1手繪

手繪結構設計

先想清楚、再動軟體
🎯 教學目標

讓學生在進入電腦繪圖前，先以手繪方式構思放大版電子鋼琴外盒的結構與造型，建立「先想清楚、再動軟體」的設計習慣，學會用草圖記錄尺寸、機構配置與外觀構想。

為什麼先手繪：手繪不需開機、修改快速，最適合在發想初期快速嘗試多種造型與配置。先在紙上把「裡面要放什麼、外面長什麼樣」想清楚，後面用 Tinkercad 建模才不會邊做邊卡。

盤點與量測

用尺或游標卡尺量出每個內部零件（音樂模組、Arduino、電池座）的長寬高，列成清單；這份尺寸表是後續手繪與建模共用的依據。

畫三視圖

在方格紙上以固定比例（1:1 或 1:2）畫出外盒的正視、側視、俯視三個角度，幫助從不同方向想像立體外盒。

標註尺寸與機構配置

標出整體尺寸、壁厚，以及喇叭出音孔、USB 出線孔、電源開關、按鈕琴鍵的位置；用箭頭或文字註記哪裡要留孔、哪裡要卡榫。

發想在地文化造型

把新營在地意象（鐵道、糖廠煙囪、地方花卉）畫成外盒表面的裝飾草圖，作為後續向量繪圖與建模的造型依據。

★ 教學重點

強調「設計始於草圖」——手繪是把腦中構想具體化的第一步，也是最快、成本最低的修改方式，培養學生動軟體前先想清楚結構與比例的設計素養。

設計 2平面

電腦繪製平面圖

從「大概」進到「準確」
🎯 教學目標

將手繪草圖轉為精確的電腦平面圖，運用 Illustrator 以正確尺寸繪製外盒各面的展開圖與開孔位置，產出可供雷切或匯入 3D 建模的向量檔案。

為什麼要畫平面圖：手繪草圖有了構想，但尺寸不夠精確。電腦平面圖能輸入毫米級的正確數值，畫出來的孔位、間距都能直接對應實體零件，是手繪與 3D 建模之間的精確化橋樑。

設定 1:1 畫板

依手繪外盒尺寸新建文件並把畫板設為實際大小（單位 mm），確保螢幕上 1 格＝實體 1mm，後續繪圖即所見即所得。

繪製各面輪廓

以矩形工具畫出底板、側板、面板等各面，用「變形面板」輸入精確長寬；複雜輪廓改用鋼筆工具。各面可攤平排列為展開圖方便雷切。

繪製開孔與標註

在面板對應位置畫出喇叭出音孔、USB 出線孔、開關開口與按鈕琴鍵孔，輸入精確直徑與座標；以文字標註關鍵尺寸，讓圖檔同時是加工說明圖。

圖層整理與輸出

依「切割線／刻線／標註」分圖層管理，切割線設細線；完成後輸出 SVG（供 Tinkercad 拉伸）或 PDF（供雷切）。

★ 教學重點

強調「精確」是平面圖的核心價值——從手繪的「大概」進到電腦的「準確」，學生體會數位工具如何把構想轉為可被機器讀取、製造的規格檔案。

設計 3建模

3D Tinkercad 建模

形隨機能 Form follows function
🎯 教學目標

以 Tinkercad 將前兩節的手繪與平面設計轉為 3D 立體外盒模型，理解「形隨機能」——先盤點內部機構所需空間，再決定外盒尺寸、壁厚與開孔，最後匯出 STL、切片列印，走完從數位設計到實體成形的完整流程。

形

什麼是「形隨機能」？

外盒不是先決定好看的造型再硬塞零件，而是先量好機構要佔多少空間，再讓造型「長」在這些機能之上。先機能、後造型，才不會做出裝不進去的漂亮空盒。

盤點零件、決定外盒尺寸

量出音樂模組、Arduino、電池座的長寬高；外盒「內部淨空間」需比零件總和再加 2～3mm 餘裕（放線材與緩衝）。內部淨空間加上壁厚，即為外部尺寸。

教學提示：先在紙上畫零件擺放配置（俯視圖），決定誰放底層、誰疊上層，再開始建模。

建立盒身與開孔

盒身：拖入立方體設為外部尺寸，再複製一個縮小（長寬各減兩倍壁厚）設為「孔洞」，群組挖出中空盒身。上蓋：底面加一圈比盒口內緣略小的凸緣（公差約 0.3～0.4mm），讓蓋子能蓋合又不卡死。機能開孔：以孔洞挖出喇叭孔、USB 孔、開關開口、琴鍵孔。

壁厚、卡榫與在地造型

壁厚：教學作品 2～3mm 即足夠堅固又省料。固定：盒內加 3D 列印固定柱（PETG）鎖電路板；上蓋用凸點卡榫或螺絲。在地元素：將新營意象轉化為外盒鏤空圖案或表面浮雕。

匯出 STL、切片與列印

盒身與上蓋分別匯出 STL（檔名如「電子鋼琴_盒身_v1.stl」）；Bambu Studio 平整面朝下擺放，外觀件層高 0.16～0.2mm、填充 10～15%、外殼 2～3 層；鏤空與懸空處檢查支撐，列印後對照零件試裝記錄需修改處。

❓ 教學提問引導

如果外盒列印出來零件裝不進去，問題比較可能出在「量測」「壁厚」還是「公差」？引導學生回到步驟一檢查淨空間餘裕，體會「設計→製作→測試→修正」的迭代循環。

★ 教學重點

強調「先機能、後造型」——外盒尺寸由內部機構決定，造型則承載美感與在地文化。引導學生體會自製外盒與第一週現成木盒的差異：木盒是既有的、要去配合它；自製外盒則反過來讓盒子配合自己的作品，這正是從「使用者」邁向「設計者」的關鍵一步。

營

在地文化延伸 · 新營意象轉譯

把地方記憶變成外盒造型

引導學生蒐集新營的代表意象——糖廠紅磚煙囪、五分車鐵道、地方花卉、舊火車站立面，並思考如何把它們「簡化」成適合雷切鏤空或 3D 浮雕的線條。重點不是寫實，而是抓住意象的輪廓特徵，讓每台鋼琴都帶有「屬於新營」的識別。

雷切篇檔案整理 → Beam Studio → 完成加工
雷切 1整檔

Illustrator 檔案整理

機器只認「線」
🎯 教學目標

在送入雷切前，學會把 Illustrator 圖檔整理成雷切機能正確讀取的格式——確認路徑封閉、線條設定正確、尺寸 1:1、文字轉外框、排版省料，避免機器停頓、重複切割或切錯。

為什麼要整理檔案：雷切機只看「線」——線是否封閉、什麼顏色、多粗，直接決定它怎麼切。送圖前花幾分鐘把檔案整理乾淨，能省下大量切壞重做的材料與時間。

檢查路徑封閉

按 Ctrl + Y 切換輪廓檢視，確認每個要切穿的形狀都是「頭尾相連的封閉路徑」；未封閉可用 Ctrl + J 連接端點。

線條顏色與粗細

切割線統一設為紅色、筆畫 0.01mm（髮絲線），讓機器辨識為「切穿」；雕刻區（如在地文化圖案）以另一顏色區分，後續在 Beam Studio 指定為雕刻。

文字外框與尺寸確認

面板含文字時全選按 Ctrl + Shift + O 建立外框（轉向量），避免雷切機無法辨識字型；再次確認畫板與零件為 1:1 實際尺寸。

排版省料與輸出

把所有零件緊密排列於材料範圍內、共用邊線可節省木板；相同零件群組複製。整理完成後輸出 SVG 或 DXF。

❓ 教學提問引導

如果雷切時某條線沒被切到，最可能的原因是什麼？引導學生回頭檢查該路徑是否封閉、線條顏色與粗細是否設定正確。

★ 教學重點

強調「機器只認線」——封閉路徑、顏色、粗細是雷切的三大關鍵；建立「送圖前先整檔」的專業習慣。

雷切 2軟體

Beam Studio 軟體教學

顏色 / 圖層即加工指令
🎯 教學目標

認識雷射切割控制軟體 Beam Studio 的操作介面，學會匯入整理好的圖檔、設定切割與雕刻參數、排版省料，並完成從電腦送圖到機器切割的完整流程。

什麼是 Beam Studio：FLUX 雷切機的控制軟體，負責把電腦裡的向量圖轉成雷射機的切割路徑——決定哪條線要「切穿」、哪塊面要「雕刻」，以及用多大功率與速度。

匯入圖檔

把整理好的 SVG／DXF 拖入工作區；匯入後確認尺寸為 1:1，避免被軟體自動縮放。

區分切割與雕刻

以線條顏色或圖層指定加工方式——封閉輪廓設為「切割（Cut）」切穿木板；表面圖案設為「雕刻（Engrave）」只燒紋路不切穿。在地文化裝飾通常用雕刻呈現。

設定參數

選擇實際材料與厚度（如 3mm 椴木合板），軟體帶出建議功率（Power）與速度（Speed）；切割用高功率低速、雕刻用低功率高速。新材料先在邊角試切確認參數。

對焦、定位與切割

放入材料執行對焦（調整雷射頭與材料距離），於預覽框確認圖檔落在材料範圍內、避開節疤；按下開始前確認排煙與防火措施，全程不可離開機器。

⚠

雷射切割安全提醒

  • 雷切過程務必開啟排煙、備妥滅火工具，操作者全程在場監看。
  • 切勿切割 PVC 等含氯材料（會釋放有毒氣體）。
  • 先在邊角「試切」確認參數，再正式切割。
★ 教學重點

強調「顏色／圖層即加工指令」——同一張圖，靠圖層區分切割與雕刻；並建立「先試切、再正式切」與全程監看的雷切安全習慣。

雷切 3加工

雷切完成 + 加工

假組先於上膠
🎯 教學目標

完成雷切後的取件、檢驗與後加工，學會以「假組」驗證零件是否吻合，並進行毛邊修整、表面處理與接合，把一堆平面木片組裝成完整的電子鋼琴外盒。

為什麼要先假組：雷切難免有誤差，先不上膠試拼一次，能在還能修改的階段發現卡榫太緊、孔位偏移等問題，避免上膠後無法挽回。

取件與檢查

從雷切機取出所有零件對照圖檔清點；檢查邊緣是否切透、有無燒焦過深或未切斷處，未切斷的可用美工刀小心分離。

假組試拼

不上膠、依組裝順序把各片卡榫嵌合，確認盒身、上蓋、面板能順利接合；卡榫過緊以砂紙微修，過鬆則記下回 Illustrator 修正公差。

毛邊修整與打磨

用砂紙（先粗後細）磨除邊緣毛刺與燒焦痕；雷切焦邊可用濕布或橡皮擦輕拭。打磨時配戴口罩、於通風處進行。

正式組裝與整合

確認假組無誤後，於卡榫接合處上白膠正式黏合；裝入 Arduino、音樂模組與電池座，按鈕對齊面板開孔，最後蓋上上蓋完成整台電子鋼琴。

❓ 教學提問引導

假組時若發現卡榫卡不進去，該回頭修「圖檔公差」還是直接「砂磨零件」？引導學生理解：偶發一兩片用砂磨即可，若每片都卡則是設計公差問題，需回 Illustrator 修正後重切。

★ 教學重點

強調「假組先於上膠」的工程習慣——可逆的試誤是控制品質的關鍵；呼應第二週雷切與本週外盒設計，完成從平面圖檔到立體成品的最後一哩。
